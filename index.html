Kopao Inventory

2022

Kopao Inventory is a website that is useful in the process of storing goods in a coffee shop called Kopao Coffe. created using the Codeigniter 3 framework and MySQL database. this website has 3 users, namely the admin, cashier at each branch of the coffee shop and the owner of the coffee shop. The admin can add, delete, edit items to each branch of the coffe shop, manage accounts, and approve for each order of goods from the coffe shop branch. for branch cashier users can request goods to the admin and record incoming goods and outgoing goods. while the owner can see reports on incoming goods and outgoing goods from each branch of the coffe shop.
